Objetivo da página

No curso, todos os projetos de desenvolvimento web realizados ao longo das aulas devem ser reunidos e documentados nesta seção. É essencial que cada projeto esteja devidamente organizado, com informações detalhadas sobre os objetivos, tecnologias utilizadas e principais funcionalidades implementadas. Além disso, conforme o aprendizado avança e novas habilidades são adquiridas, será necessário revisar e atualizar esses projetos de forma contínua, implementando melhorias ou corrigindo possíveis falhas, garantindo que reflitam as melhores práticas e o conhecimento mais recente em desenvolvimento web.

Pagina Pessoal

Minha página pessoal foi criada com o intuito de apresentar um pouco sobre mim, meus hobbies, as linguagens de programação que tenho constume de usar e também disponibilizar meus meios de contato. Utilizei HTML para estruturar todas as informações, CSS para a estilização da página e JavaScript para adicionar a funcionalidade ao botão 'Portfólio', permitindo que ele retorne à página principal do portfólio quando clicado.

Evento

A página de evento foi criada para informar sobre um evento especial de Natal que ocorrerá em Goiânia. Nela, são fornecidos detalhes como o local, a data, os cantores participantes e, mais abaixo, um botão de 'Reserva'. O usuário deve informar seu nome e e-mail para que o ingresso seja enviado para o seu e-mail. Ao final da página, estão disponíveis os meios de contato do evento e um pequeno botão para retornar ao portfólio. Para a criação da página, foram utilizados HTML, CSS e JavaScript. O HTML estruturou as informações da página, o CSS cuidou da estilização e o JavaScript foi responsável por exibir uma tela oculta até que o usuário clique em 'Reserva'. O JS também captura as informações fornecidas pelo usuário e, ao clicar em 'Confirmar', um pop-up aparece informando que a reserva foi concluída. Caso contrário, outro pop-up surge informando que a reserva foi cancelada.

Calculadora

A página da calculadora foi criada para fornecer uma interface interativa de cálculo. Ela possui uma tela de exibição onde os valores e resultados são mostrados, além de botões para números, operações matemáticas e funcionalidades como limpar e apagar. O usuário pode realizar cálculos utilizando operações comuns como adição, subtração, multiplicação, divisão, e ainda pode inserir parênteses para operações mais complexas. A interação do usuário com os botões é feita através de JavaScript, que processa os valores digitados, executa o cálculo e exibe o resultado. A funcionalidade de limpar reseta a tela para zero, e a função de backspace apaga o último caractere digitado. O cálculo é realizado com o uso da função eval, que interpreta a expressão matemática e retorna o resultado, tratando símbolos como '÷' e '×' para suas versões aritméticas. Para a construção da página, foram utilizados HTML, CSS e JavaScript. O HTML estruturou a página, definindo os elementos da interface, o CSS estilizou os botões e a tela da calculadora, e o JavaScript ficou responsável pela interação com o usuário e pelo cálculo em si.
